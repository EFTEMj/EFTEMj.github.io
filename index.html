Zeiss WinTEM - Energy Filter Align (EFA)

Zeiss WinTEM - Energy Filter Align (EFA)
  • can be found at Service > Energy Filter...
  • control of 7 filter elements
    • 4 dipole elements
      -> shift the spectrum
    • 3 quadrupole elements
      -> deform and rotate the spectrum
  • deform means to compress or expand the spectrum in lateral direction
    • deform x (QSinK7): increase the lateral size
    • deform y (QSinK4): controls the focus of this axis
    • rotation (QCosK4)

Aberrations - 2D-visualization

\(\Delta E\) represents the energy offset at the given dataset. Electrons that traverse the energy filter at an optimal ray pass will have an energy offset of \(\Delta E = 0\).

\(y_n\) is the position, where the polynomial \(f_n\) intersects the axis with \(\Delta E = 0\).

$$ \Rightarrow f_n(0) = y_n$$

$$f_n(\Delta E) = \sum^2_{i,j=0}a_{ij}\cdot y_n^j\cdot \Delta E^i$$

with \(a_i = a_{i0} + a_{i1}\cdot y_n + a_{i2}\cdot y_n^2\)

$$\Rightarrow f_n(\Delta E) = \sum^2_{i=0}a_i\cdot \Delta E^i$$

$$f_n(\Delta E) = \sum^2_{i,j=0}a_{ij}\cdot y_n^j\cdot \Delta E^i$$

Correction by image processing

Correction - Remark
Correction - Overview
  • \(\Delta E\) is shown on the horizontal axis of the SR-EELS dataset.
  • \(x\) is shown on the vertical axis of the SR-EELS dataset.
  • \(\Delta \tilde{E}\) and \(\tilde{x}\) are the related curved cooradinats.
  • \(\tilde{x}_j(\Delta E)\) describes the curve where the same lateral position is recorded on the detector.

$$\tilde{x}_j(\Delta E) = \sum_{i=0}^m\tilde{a}_{ij}\cdot \Delta E^i$$

These equations can be combined by using \(x_0 = \tilde{x}_j(\Delta E = 0)\).

$$\Rightarrow\tilde{x}(\Delta E,x_0) = \sum_{i,j=0}^{m,n}\tilde{a}_{ij}\cdot \Delta E^i\cdot x_0^j$$

Correction - The \(\Delta E\)-axis (1)

If the energy loss is affected by the curved axes, we can transform it by calculating the arc length.

$$\Delta \tilde{E} = s_{\tilde{x}}(\Delta E) = \int_0^{\Delta E} \sqrt{1+\left(f_{\tilde{x}}'(z)\right)^2}\mathrm{\ d}z$$

$$\mathrm{with\quad} a = 4a_2^2\mathrm{,\quad } b = 4a_1a_2 \mathrm{\quad and\quad } c = a_2^2+1$$

$$\mathrm{and\quad} a_h = a_{h0} + a_{h1}\cdot x_0 + a_{h2}\cdot x_0^2$$

$$\Rightarrow \Delta \tilde{E}(\Delta E) = \int_0^{\Delta E} \sqrt{a\cdot z^2+b\cdot z+c}\mathrm{\ d}z$$

Correction - The \(\Delta E\)-axis (2)

$$\Rightarrow \Delta \tilde{E}(\Delta E) = \int_0^{\Delta E} \sqrt{a\cdot z^2+b\cdot z+c}\mathrm{\ d}z$$

$$\mathrm{with\quad} g'(y) = \frac{1}{f'(x)}\quad and\quad g(y) = f^{-1}(y)$$

$$\Rightarrow \Delta E = s^{-1}(\Delta \tilde{E}) = \int_0^{\Delta \tilde{E}} \frac{\mathrm{d\,}z}{\sqrt{az^2+bz+c}}$$

Correction - The \(\Delta E\)-axis (3)

$$\Delta E(\Delta \tilde{E}) = \int_0^{\Delta \tilde{E}} \frac{\mathrm{d\,}z}{\sqrt{az^2+bz+c}}$$

$$\Rightarrow\Delta E(\Delta \tilde{E}) = \left.\frac{1}{\sqrt{a}}\arcsin^{-1}\frac{2az+b}{\sqrt{4ac-b^2}}\right|^{z=\Delta \tilde{E}}_{z=0}$$

$$\mathrm{for\quad }4ac-b^2 = 4a_2^2\cdot(7a_1^2 + 11) > 0$$

Correction - Full transformation

\(\Delta E\)-coordinate:

$$\Delta E(\Delta \tilde{E}) = \left.\frac{1}{\sqrt{a}}\arcsin^{-1}\frac{2az+b}{\sqrt{4ac-b^2}}\right|^{z=\Delta \tilde{E}}_{z=0}$$

$$= \frac{1}{\sqrt{a}}\left(\textrm{arcsinh}\frac{2\,a\cdot\Delta \tilde{E}+b}{\sqrt{4\,ac-b^2}} - \textrm{arcsinh}\frac{b}{\sqrt{4\,ac-b^2}}\right)$$

$$\mathrm{with\quad } a = 4\,a_2^2\mathrm{,\quad } b = 4\,a_1a_2 \mathrm{\quad and\quad } c = a_2^2+1$$

\(x\)-coordinate:

$$x = f_{\tilde{x}}(\Delta \tilde{E}) = a_2\cdot \Delta \tilde{E}^2+a_1\cdot \Delta \tilde{E}+a_0$$

  • The size of each pixel changes.

  • A corrected pixel will overlay several uncorrected pixels.
The intensity of the corrected pixel is made up of the partial intensities of several uncorrected pixels.

Summary
  • The imaging filter can be adjusted by several controls, but only QSinK7 (deform x) and QSinK4 (deform y) are useful.
  • The lateral magnification can be adjusted independently from the energy magnification.
  • The distortion of the imaging filter can be described by a function series of simple polynomials of 2nd degree.
  • The main problem of the correction is the mapping of calculated continuous coordinates to discrete coordinates of the recorded images.
